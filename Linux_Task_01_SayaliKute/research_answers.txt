Linux Task 01 - Research Answers

1. What is Linux?

Linux is an open-source operating system used in computers, servers, cloud platforms, and cybersecurity environments. It is known for its stability, security, and flexibility.

---

2. Why is Linux important in Cyber Security?

Linux is important in Cyber Security because most security tools, servers, and penetration-testing environments run on Linux. It provides better control, powerful command-line utilities, and enhanced security features.

---

3. Difference between Linux and Windows

Linux:

* Open-source operating system
* More secure and customizable
* Mostly used in servers and cybersecurity
* Free to use

Windows:

* Proprietary operating system
* User-friendly graphical interface
* Commonly used on personal computers
* Requires licensing

---

4. What is a Linux Distribution?

A Linux Distribution (Distro) is a version of Linux that includes the Linux kernel along with software packages and tools. Examples include Ubuntu, Kali Linux, Fedora, and Parrot OS.

---

5. Why do Ethical Hackers prefer Linux-based Operating Systems?

Ethical hackers prefer Linux because it includes powerful cybersecurity tools, offers greater control over the system, supports scripting and automation, and is widely used in professional security environments.
